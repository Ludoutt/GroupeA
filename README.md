# GroupeA

## Technologies :

* Symfony 5

## Installation :

* Clone repo
* cd /site
* composer install
* Config your .env file for your database connexion
* php bin/console d:c
* php bin/console d:m:m
* php bin/console doctrine:fixtures:load
* php bin/console server:run
