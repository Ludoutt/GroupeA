# GroupeA

## Technologies :

* Symfony 5

## Installation :

* Clone repo
* cd /app
* composer install
* php bin/console server:run

## Installation front :

* Installation de Ruby: https://rubyinstaller.org/downloads/
* commande: gem install sass
* commande: gem install compass
* npm install
